# nutrition-coach

**Claude Code skill** — первичная помощь по микронутриентам для жителей СНГ без доступа к врачу.

Skill называет вероятный дефицит, решительно трактует анализы, даёт терапевтические дозы и протоколы коррекции, предметно разбирает рецептурные препараты. База знаний построена на 8 Deep Research отчётах: свежие RCT и гайдлайны 2023–2026, эпидемиология дефицитов СНГ, лабораторная диагностика, лекарственные взаимодействия, региональная специфика.

---

## Зачем это нужно

В странах СНГ доступ к врачу-нутрициологу или грамотному терапевту — редкость. Люди гуглят симптомы, читают форумы или получают шаблонные ответы «сходите к врачу». Этот skill закрывает gap: действует как компетентный первичный приём, даёт конкретный план с дозами и формами, а врача рекомендует как дополнение, а не замену.

---

## Что умеет

- **Диагностика дефицитов** по симптомам, образу жизни и лекарствам — без анализов или по имеющимся (ферритин, 25(OH)D, B12, гомоцистеин и т.д.)
- **Терапевтические протоколы** — форма, доза, тайминг, длительность, контроль
- **Трактовка анализов** из Инвитро / Хеликс / KDL / Олимп РК с учётом функциональных vs. референсных норм
- **Лекарственные взаимодействия** — 13 групп препаратов: метформин, ИПП, статины, КОК, левотироксин, диуретики и др.
- **Спецгруппы** — беременные, дети, пожилые, веганы, спортсмены, Хашимото, СПКЯ, СД2, бариатрия, постменопауза
- **Региональная специфика** — йод/селен-дефицитные зоны, «витамин-D-зима», фортификация муки фолатом (есть в Казахстане, нет в Украине)
- **Инструкция по этикетке** — как самостоятельно выбрать форму и дозу на полке без посредника

## Как активируется

Skill активируется при любом из сигналов — даже без слова «нутрициология»:

- Жалобы на **утомляемость, выпадение волос, крампи, парестезии, депрессию** на фоне образа жизни / лекарств / диеты / беременности
- Вопросы про **витамины, минералы, БАДы, дефициты**
- Просьбы **расшифровать анализы** (ферритин, 25(OH)D, B12, гомоцистеин и т.д.)
- Вопросы про **влияние лекарств** на нутриентный статус
- Упоминание **беременности, веганства, Хашимото, СПКЯ, бариатрии, спорт-питания**

## Принципы работы

**Три уровня безопасности:**
- 🟢 Уровень 1 — профилактические дозы в пределах UL: выдаются сразу, без трения
- 🟡 Уровень 2 — терапевтические дозы и рецептурные: выдаются с одним встроенным условием (анализ/порог), но не передаются врачу
- 🔴 Уровень 3 — истинные неотложки: эскалация в скорую/приёмный покой

**Контракт ответа:** каждый ответ содержит (1) прямую гипотезу, (2) уровень уверенности, (3) план с дозами и таймингом, (4) что делать сегодня, (5) один триггер эскалации.

Skill никогда не заканчивает ответ единственной фразой «сходите к врачу» — для аудитории без доступа к врачу это оставление без помощи.

## База знаний

Skill построен на основе 8 Deep Research отчётов, охватывающих:

| Тема | Источник |
|------|----------|
| Эпидемиология дефицитов в СНГ | Систематические обзоры, национальные нутриентные исследования |
| RCT и гайдлайны 2023–2026 | Endocrine Society, EFSA, WHO, ESPGHAN, журналы NEJM/Lancet/JAMA |
| Формы и дозы | Кокрейн, фармакокинетические исследования |
| Лекарственные взаимодействия | Drug-nutrient interaction databases, клинические руководства |
| Лабораторная диагностика | Референсные интервалы Инвитро / Хеликс / KDL / Олимп РК |
| Региональная специфика | Эндемические карты йода/селена, данные о фортификации |
| Lifestyle-маркеры | Когортные исследования с OR/aOR по 12 нутриентам |
| Спецгруппы | Клинические рекомендации по беременности, педиатрии, гериатрии |
