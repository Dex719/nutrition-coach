# Скрининговый опросник нутритивного статуса

> **Назначение:** Используй этот опросник, чтобы собрать структурированный контекст по пользователю. Не вываливай всё сразу — иди блоками, по 3-5 вопросов за раз, простым человеческим языком (термины объясняй). Режимов A/B/C нет — пользователь один (см. «Кто ты и как ты работаешь» в SKILL.md). Полезно дополнительно уточнить: доступ к аптеке/лаборатории и регион СНГ — это влияет на реалистичность плана.

---

## Блок 1: Базовые данные (всегда нужны)

   - Менструальный цикл регулярный / нерегулярный / отсутствует (постменопауза, лактация и т.д.)
   - Беременность сейчас: нет / да (срок: ___)
   - Лактация: нет / да (возраст ребёнка: ___)
   - Планирование беременности в ближайший год: нет / да
4. **Страна и регион проживания:**
   - Россия (укажи область/город) / Казахстан (область/город) / Беларусь (область) / Украина (область) / Узбекистан (область) / другое: ___
5. **Рост (см):** ___ **Вес (кг):** ___
6. **Профессия и характер работы:** офис / физический труд / в основном дома / много на улице / другое: ___

## Блок 2: Симптомы и жалобы

Что из перечисленного есть последние 3+ месяцев? (можно несколько)

### Энергия и общее состояние
- [ ] Постоянная утомляемость, не проходит после отдыха
- [ ] Сонливость днём, трудно проснуться утром
- [ ] Снижение выносливости при привычной нагрузке
- [ ] Бледность кожи, одышка при подъёме по лестнице

### Нервная система
- [ ] Тревожность, раздражительность
- [ ] Депрессивное настроение, апатия
- [ ] Трудности концентрации, «туман в голове»
- [ ] Парестезии (онемение/покалывание пальцев)
- [ ] Шаткость походки, ухудшение памяти
- [ ] Бессонница, поверхностный сон, частые пробуждения
- [ ] Кошмарные сновидения (особенно на фоне лекарств)

### Мышцы и кости
- [ ] Мышечные крампи, особенно ночью в икрах
- [ ] Подёргивание века
- [ ] Боли в мышцах при нагрузке (если на статинах — критично)
- [ ] Боли в крупных суставах
- [ ] Переломы при минимальной травме в анамнезе

### Кожа, волосы, ногти
- [ ] Выпадение волос
- [ ] Ломкость, расслоение ногтей
- [ ] Белые пятна на ногтях
- [ ] Заеды в углах рта (хейлит)
- [ ] Жжение языка (глоссит)
- [ ] Сухость кожи, шелушение
- [ ] Себорейный дерматит

### Иммунитет
- [ ] Частые простуды (>4 в год)
- [ ] Долго не заживают ранки
- [ ] Частые герпетические высыпания

### ЖКТ
- [ ] Изжога регулярно (на её фоне приём ИПП?)
- [ ] Запоры регулярные
- [ ] Жидкий стул регулярно
- [ ] Вздутие живота
- [ ] Жирный блеск стула (мальабсорбция жиров)

### Сердечно-сосудистая
- [ ] Сердцебиение, перебои в сердце
- [ ] Ортостатическое головокружение
- [ ] Отёки ног

### Эндокринная
- [ ] Зябкость, плохо переносит холод
- [ ] Сухость кожи + запоры + утомляемость (триада гипотиреоза)
- [ ] Узлы в щитовидной железе на УЗИ
- [ ] Нерегулярный цикл, бесплодие, акне (СПКЯ?)
- [ ] Дисгевзия (искажение вкуса) — особенно на фоне иАПФ

### Прочее
- [ ] Тяга к необычным продуктам (лёд, мел, земля — пикацизм при дефиците Fe)
- [ ] Изменение вкуса
- [ ] Снижение либидо

## Что делать с заполненным опросником

После заполнения:

1. **Сопоставь симптомы** с таблицей в `screening_algorithm.md` → получи список нутриентов в фокусе.

2. **Учти все лекарства** — открой `protocols/by_medication.md` и собери список истощаемых нутриентов.

3. **Определи спецгруппы** — открой `protocols/by_population.md` (беременность, пожилые, веганы и т.д.).

4. **Проверь red flags** (см. `screening_algorithm.md` Шаг 4).

5. **Сформулируй рекомендацию по Контракту ответа:**
   - Наиболее вероятная причина прямо + уверенность одним словом
   - Конкретный план: нутриент, форма, доза, тайминг (включая то, что можно начать без анализов)
   - Какие анализы уточнят/скорректируют (приоритезированный список); для опасных нутриентов — встроенное предусловие Уровня 2
   - Что делать сегодня + один триггер эскалации (Уровень 3)

6. **Установи timeline пересмотра:**
   - Через 4-6 недель: первичная оценка эффекта lifestyle
   - Через 8-12 недель: переоценка после анализов и коррекции
   - Через 3-6 месяцев: полный пересмотр

## Сокращённая версия (для быстрого скрининга)

Если время ограничено — 10 ключевых вопросов:

1. Возраст, пол, регион СНГ
2. Беременность/лактация (если применимо)
3. Принимаете ли регулярно ИПП, метформин, статины, КОК или диуретики?
4. Веганство или строгое вегетарианство?
5. Жирная морская рыба чаще 2 раз/нед?
6. Молочные продукты ежедневно?
7. Менструации обильные? (для женщин)
8. Симптомы: утомляемость, выпадение волос, крампи, парестезии, депрессия?
9. Хронические заболевания: ЩЖ, СД, ХБП, ВЗК?
10. Имеющиеся анализы (особенно 25(OH)D, ферритин, B12) — за последний год?

По этим ответам можно дать первичную ориентировку и определить, какие нутриенты в первую очередь смотреть.
