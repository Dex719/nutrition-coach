# Витамины B1 (Тиамин) и B6 (Пиридоксин) — cheat-sheet

Объединённый файл по B1 и B6 — у обоих много пересечений по группам риска (алкоголь, диуретики, КОК).

---

## Витамин B6 (Пиридоксин)

### Cut-offs

| Маркер | Норма | Дефицит |
|---|---|---|
| **PLP (пиридоксаль-5'-фосфат)** — активная форма | 30–110 нмоль/л | <20 нмоль/л |
| Общий B6 сыворотки | вариабельно | менее точен, чем PLP |

### Lifestyle-маркеры B6

| Фактор | Сила |
|---|---|
| **КОК без B-комплекса** | 78% дефицит PLP <20 (vs 25% без КОК) |
| Алкоголь регулярно | >50% алкоголиков дефицитны |
| ИМТ ≥30 или <18.5 | β=-2.7 (p=0.034) |
| Азиатские этнические группы | -21.1% PLP |
| Изониазид, леводопа, теофиллин, гидралазин | связывают PLP |
| Аутоиммунные (РА, ВЗК) | системное воспаление ↑ катаболизма PLP |
| Гемодиализ | вымывание |
| Курение | значимое ↓ |
| Депрессия, раздражительность, себорейный дерматит | классические маски |

### Формы и дозы B6

- **Пиридоксаль-5'-фосфат (P5P)** — активная коферментная форма, лучший выбор при КОК, заболеваниях печени
- Пиридоксина гидрохлорид — стандарт, требует конверсии в P5P

**Дозы:**
- RDA: 1.3-1.7 мг/сут
- Профилактика: 2-5 мг/сут (особенно при КОК)
- При изониазиде: 25-50 мг/сут (обязательно!)
- При ПМС, тошноте беременных: 25-50 мг (с магнием)
- При выраженной нейропатии: 50-100 мг
- **UL: 100 мг/сут!** Длительное превышение → периферическая сенсорная нейропатия (парадокс — лечим нейропатию, получаем её обратно)

### Лекарственные взаимодействия B6

| Препарат | Эффект |
|---|---|
| **КОК** | ↓ PLP, обязательная дотация 2-5 мг/сут |
| **Изониазид** (туберкулёз) | блокирует PLP, обязательная дотация 25-50 мг |
| Леводопа (без карбидопы) | B6 ускоряет периферическое декарбоксилирование, ↓ эффект леводопы |
| Леводопа + карбидопа | взаимодействие минимально, B6 безопасен |
| Гидралазин | связывает PLP |
| Пеницилламин | связывает PLP |

### Алгоритм B6

1. Высокий риск: КОК, изониазид, алкоголь, депрессия, ПМС
2. Заказать PLP (в СНГ не везде доступен; общий B6 как альтернатива)
3. Дотация P5P 2-5 мг/сут (профилактика) или 25-50 мг (терапия)
4. ⚠️ Не превышать 100 мг/сут длительно

## Витамин B1 (Тиамин)

### Cut-offs

| Маркер | Норма | Дефицит |
|---|---|---|
| **Тиамин-пирофосфат эритроцитов (TPP)** — функциональный | 70–180 нмоль/л | <70 |
| Транскетолаза эритроцитов | по активности | косвенный маркер |
| Тиамин сыворотки | менее точен | устаревший метод |

### Lifestyle-маркеры B1

| Фактор | Сила |
|---|---|
| **Алкоголизм** | до **80% имеют дефицит** (риск энцефалопатии Вернике) |
| Крепкий чай / кофе >470 мл/сут | тиаминазы разрушают B1 в ЖКТ |
| Избыток сахара/рафинированных углеводов | ↑ расход B1 на окисление глюкозы |
| Белый шлифованный рис ≥4 раз/нед | базовое ↓ (рис теряет 90% B1 при шлифовке) |
| Возраст >60 | aOR=1.98 |
| Женский пол | aOR=2.13 |
| Петлевые диуретики (фуросемид) | вымывание с мочой |
| Бариатрия, хроническая рвота | быстрое истощение |
| Гестационный токсикоз с рвотой | риск Вернике |
| ХСН на фуросемиде с нарастающей одышкой | МАСКА B1-дефицита (миокардиоэнергодефицит) |

### Формы

- **Бенфотиамин** — жирорастворимая форма, в 5x раз лучше абсорбируется, проникает в нерв, лучшая для диабетической нейропатии
- **Сульбутиамин** — синтетическая липофильная, для когнитивных задач
- Тиамина гидрохлорид — стандарт, водорастворимый

### Дозы

- RDA: 1.1-1.2 мг/сут
- Профилактика: 5-10 мг/сут (мультикомплекс)
- При фуросемиде: **50-100 мг/сут** (обязательно!)
- При алкоголизме / Вернике-Корсакове: 100-500 мг в/м или в/в (никогда после глюкозы — глюкоза разрушит остатки B1 и спровоцирует Вернике!)
- При диабетической нейропатии: бенфотиамин 150-300 мг/сут
- UL не установлен (быстро выводится)

### Лекарственные взаимодействия

| Препарат | Эффект |
|---|---|
| **Петлевые диуретики (фуросемид)** | вымывание, обязательная дотация 50-100 мг |
| Хронический алкоголь | блокада абсорбции и фосфорилирования |
| Высокие дозы кофе/чая (тиаминазы) | разрушение в ЖКТ |
| Антациды длительно | ↓ всасывания |

### Алгоритм B1

1. Высокий риск: алкоголь, фуросемид, ХСН с одышкой, диабет с нейропатией, бариатрия
2. Заказать TPP эритроцитов (если доступен) или эмпирическая дотация
3. При фуросемиде: автоматически 50-100 мг тиамина/сут
4. При диабетической нейропатии: бенфотиамин 150-300 мг 3-6 мес
5. При алкоголизме (госпитализация): парентерально, ДО глюкозы
