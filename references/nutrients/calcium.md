# Кальций (Ca) — cheat-sheet

## Cut-offs

| Маркер | Норма |
|---|---|
| Общий кальций сыворотки | 2.15–2.55 ммоль/л |
| Ионизированный (биологически активный) Ca | 1.15–1.35 ммоль/л |
| Паратгормон (ПТГ) | 15–65 пг/мл (косвенный — ↑ при дефиците Ca/D) |
| Суточная кальциурия | 2.5–7.5 ммоль/сут |

⚠️ **Сывороточный Ca плохой маркер питания** — гомеостаз жёстко поддерживается ПТГ за счёт вымывания из костей. Дефицит диеты не виден в анализе, но виден через:
- ↑ ПТГ (вторичный гиперпаратиреоз)
- Снижение МПК (денситометрия)

## Lifestyle-маркеры

| Фактор | OR |
|---|---|
| ИПП + Ca-добавки одновременно | OR=1.56 (1.08-2.23) для вторичного гиперпаратиреоза |
| Избыток натрия (соленья, полуфабрикаты) | +2.5 г соли = -26.3 мг Ca с мочой |
| ≥3 чашек кофе/сут + низкое потребление молочных | OR=3.59 (1.46-8.85) для переломов бедра |
| Молочные <1 порция/день | гарантированный отрицательный баланс |
| Женщины в постменопаузе | критическая группа |
| Девочки-подростки | дефицит у 81% |
| Кортикостероиды | aOR=2.09 для дефицита |
| Курение | прямое токсическое на остеобласты |
| Дефицит витамина D | всасывание ↓ с 45% до <15% |

## Формы

| Форма | Биодоступность | Когда |
|---|---|---|
| **Цитрат** | высокая, не требует кислотности | универсал, особенно при ИПП, у пожилых |
| **Карбонат** | средняя, требует HCl | дёшево, только с едой |
| **Гидроксиапатит** | высокая | + микроэлементы кости |
| **Лактат, глюконат** | средняя | в детских формах |
| Из пищи (молочные, зелень, миндаль, сардины с костями) | средне-высокая | первый выбор всегда |

## Дозы

- **RDA:** 1000-1200 мг/сут (взрослые), 1200 мг (>50 лет женщины, >70 мужчины), 1300 мг (подростки)
- **Профилактика остеопороза:** 1000 мг/сут (если из пищи <500-700 мг)
- **При кортикостероидах:** 1000-1500 мг + 1000-2000 МЕ D с первого дня
- **При антиконвульсантах:** 1000 мг + 2000-4000 МЕ D
- **Беременность:** 1000-1300 мг/сут
- **UL:** 2500 мг/сут

⚠️ **Не превышать дозу 500-600 мг за один приём** — выше этого % абсорбции резко падает. Разделять на 2-3 приёма.

## Сочетания

- **Карбонат — строго с едой** (требует HCl)
- **Цитрат — можно натощак** или с едой
- **Принимать отдельно от:** Fe, Zn, Mg в высоких дозах (конкуренция за DMT-1) — интервал 2 ч
- **Левотироксин** — интервал 4 часа
- **С витамином D + K2 + Mg** для правильного распределения (не в сосуды, а в кости)

## Лекарственные взаимодействия

| Препарат | Эффект |
|---|---|
| ИПП | ↓ всасывания карбоната, цитрат не страдает |
| Кортикостероиды | ↑ потери, нужна дотация с первого дня |
| Антиконвульсанты | ↑ деградации витамина D → косвенно ↓ Ca-абсорбции |
| Левотироксин | хелатирование при одновременном приёме |
| Тиазидные диуретики | задерживают Ca (синергия при дотации) |
| Петлевые диуретики | ↑ экскреции Ca |

## Red flags

- **Гиперкальциемия любой этиологии** (саркоидоз, гиперпаратиреоз, опухоли) — Ca противопоказан
- **Камни в почках кальциевые** — осторожность, нужна нормальная гидратация и контроль кальциурии
- **Высокие дозы Ca без K2 + Mg при дефиците D** — риск кальцификации сосудов
- **Болезнь Педжета костей**
- **Совместный приём с тетрациклинами, фторхинолонами** — хелатирование

## Спецгруппы

- **Постменопауза:** Ca 1000-1200 + D 1000-2000 МЕ + K2 + физ. нагрузка
- **Подростки:** 1300 мг/сут, чаще через молочные
- **Беременность:** 1000-1300 мг/сут
- **На кортикостероидах:** 1000-1500 мг + D с первого дня
- **Лактозная непереносимость:** альтернативы — твёрдые сыры, йогурт (хорошо переносятся), кунжут, сардины

## Алгоритм

1. Спросить: возраст, пол, молочные в рационе, кофе, соль, лекарства, переломы в анамнезе
2. Если возраст +50 / постменопауза / кортикостероиды / переломы — оценить кальциемию + ПТГ + 25(OH)D + денситометрия
3. **Сначала корректировать витамин D** — без него любая дотация Ca малоэффективна
4. Из пищи: молочные 3 порции/день (если переносит)
5. Если нужна добавка: цитрат 500 мг 2 раза/день с едой + K2 90-120 мкг + Mg
6. Снижать соль, кофе, бросать курить
