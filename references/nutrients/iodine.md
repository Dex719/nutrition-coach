# Йод (I) — cheat-sheet

## Cut-offs (медиана йодурии — ТОЛЬКО для популяционных исследований, не для индивидуальной диагностики!)

| Медиана UIC (мкг/л) | Интерпретация |
|---|---|
| <20 | тяжёлый дефицит |
| 20–49 | умеренный |
| 50–99 | лёгкий |
| **100–199** | **адекватный** |
| 200–299 | избыток (риск аутоиммунных тиреопатий) |
| ≥300 | избыточное потребление |

**Для беременных:** адекватная медиана 150–249 мкг/л.

⚠️ **Разовая йодурия не подходит для индивидуальной диагностики** — вариабельность огромная, отражает только последние 24-48 ч. РАЭ и ВОЗ: использовать только в эпидемиологии.

Для индивидуальной оценки: УЗИ щитовидной железы + ТТГ + тиреоглобулин (косвенный маркер хронического дефицита).

Источник: `lab_diagnostics.md` § 7.

## Распространённость в СНГ

- **РФ:** среднесуточное потребление 40-80 мкг/сут (в 3 раза ниже нормы). 95% детей и 65% взрослых имеют тиреоидную патологию, связанную с дефицитом
- **Украина (Житомирская обл.) у беременных:** медиана UIC = **13 мкг/л** (тяжелейший дефицит, 95.9% имеют <150)
- **РФ (Кавказ, Алтай, Тыва, Якутия, Урал, Сибирь):** mUIC школьников 70-90 (лёгкий дефицит)
- **Узбекистан:** в семьях используется адекватно йодированная соль только у 36% (формально обязательно)
- **Казахстан:** mUIC школьников 150-180 (адекватный) — благодаря обязательному йодированию с 2003
- **Беларусь:** mUIC 110-130 (адекватный) — обязательное промышленное йодирование

Источник: `regional_specifics.md` § 1.

## Lifestyle-маркеры

| Фактор | OR |
|---|---|
| Использование нейодированной соли (включая морскую, гималайскую розовую) | дефицит у 40.4% |
| Хранение йодированной соли >2 мес после вскрытия | aOR=2.66 |
| Хранение соли в открытой таре у плиты | OR=3.85 (1.88-7.87) |
| Добавление соли в начале варки (термический распад йода) | aOR=0.41 при добавлении в конце |
| Жёсткое ограничение соли по медпоказаниям (женщины) | aOR=1.79 |
| Морская рыба/морепродукты <1 раз/нед | критично для континентальных регионов |
| Сырые крестоцветные ежедневно (капуста, брокколи, шпинат, соя) | гоитрогены блокируют захват йода |
| Беременность без препаратов KI | риск кретинизма у плода |
| Утомляемость, сухость кожи, зябкость, запоры | классика субклинического гипотиреоза |

Источник: `lifestyle_markers.md` § Йод.

## Формы

| Форма | Когда |
|---|---|
| **Калия йодид (KI)** | стандарт, в препаратах (Йодомарин, Йодбаланс) и в соли |
| **Калия йодат (KIO₃)** | в современной йодированной соли — стабильнее KI при хранении |
| Молекулярный йод (Lugol) | не рекомендуется для самолечения |
| Из ламинарии | вариабельная концентрация, риск передозировки |

## Дозы

- **RDA взрослые:** 150 мкг/сут
- **Беременность:** 220-250 мкг/сут (по ВОЗ), РФ нормативы — 220 мкг
- **Лактация:** 290 мкг/сут
- **Дети 7-12 лет:** 120-150 мкг/сут
- **UL для взрослых:** 1100 мкг/сут (ВОЗ) — превышение запускает йод-индуцированный гипотиреоз или Базедов
- **Терапевтически при эндемическом зобе:** под наблюдением эндокринолога, обычно 100-200 мкг

⚠️ **При узловом зобе с функциональной автономией доза >200 мкг противопоказана** до исключения автономии (риск йод-индуцированного тиреотоксикоза, феномен Йода-Базедова).

## Что искажает йодурию

- **Йодсодержащие антисептики** (раствор Люголя, Бетадин) на коже/слизистых → ↑↑↑ йодурии на недели вперёд
- **Контрасты КТ/МРТ**
- **Амиодарон** (содержит йод)
- **Ламинария, морепродукты накануне** — отменить за 3 дня
- **Любые мультивитамины с KI** — отменить за 3 дня

## Йод + Селен (критическая синергия!)

При сочетанном дефиците йода и селена изолированная дотация йода может спровоцировать **прогрессирование аутоиммунного тиреоидита Хашимото** — активация тиреопероксидазы создаёт перекись H₂O₂, которая не утилизируется глутатионпероксидазами без селена.

В эндемичных по селену регионах (Северо-Запад РФ, Полесье, Северный Казахстан, Бурятия) йод обязательно сочетать с селеном 100-200 мкг/сут.

Источник: `regional_specifics.md` § 2.

## Спецгруппы

- **Беременные:** обязательная дотация 150-250 мкг с этапа планирования (профилактика кретинизма)
- **Хашимото:** йод осторожно, ВСЕГДА с селеном, под контролем АТ-ТПО
- **Узловой зоб:** только под контролем эндокринолога
- **Дети:** см. `special_populations.md` § 2

## Алгоритм

1. Спросить: какая соль на кухне (йодированная или нет), как хранится, когда солит еду, частота морской рыбы, симптомы гипотиреоза
2. Если регион СНГ + симптомы → заказать ТТГ + Т4св + АТ-ТПО + УЗИ ЩЖ
3. Йодурия — только эпидемиологически или для исключения избытка
4. Профилактика взрослым: йодированная соль + 100-150 мкг KI/сут в осенне-зимний период
5. Беременным: 150-250 мкг KI/сут обязательно
6. В селенодефицитных регионах: + 100-200 мкг селенометионина
