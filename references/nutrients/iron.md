# Железо (Fe) — cheat-sheet

## Cut-offs ферритина (главный маркер запасов)

| Уровень | нг/мл | Интерпретация |
|---|---|---|
| Абсолютный дефицит (новые нормы BSH, РАЭ) | <30 | требует коррекции |
| Старые нормы (устарели) | <15 | использовать НЕ нужно |
| При воспалении (СРБ↑) — порог сдвигается | <100 | трактовать как дефицит |
| Функциональная медицина: оптимум | 50–70 + (как идеальный вес в кг) | спорно, но указывать |
| Беременность III триместр | <30 + контроль ОЖСС | критично |

**Маркеры 2-й линии:**
- % насыщения трансферрина: норма 30–40%, <16% — дефицит
- sTfR (растворимые рецепторы трансферрина): не острофазовый, диагностика при воспалении
- sTfR-F index = sTfR / log₁₀(ферритин): <1 АХЗ, 1-2 смешанный, >2 истинный дефицит
- ОАК с эритроцитарными индексами (MCV, MCH, RDW)

Источник: `lab_diagnostics.md` § 1.

## Формы

| Форма | Биодоступность | Переносимость | Когда |
|---|---|---|---|
| **Бисглицинат** | высокая | отличная | первый выбор, особенно у женщин с чувствительным ЖКТ |
| Сульфат (сорбифер, актиферрин) | 25-30% | низкая (запоры, тошнота) | классика, дёшево |
| Фумарат | средняя | средняя | альтернатива сульфату |
| Липосомальное (sucrosomial) | высокая | отличная | НПВП, ВЗК, бариатрия |
| Гидроксид полимальтозат (Fe³⁺) | средняя | хорошая | у детей |
| Гемовое (печень, мясо) | 25-30% | — | пища, без проблем |
| Негемовое (растения) | 2-8% | — | пища, с витамином C для усиления |

## Дозы

- **Профилактика** (планирование беременности, обильные менструации): 25 мг элементарного Fe/сут бисглицинат
- **Латентный дефицит** (ферритин <30): 50-100 мг/сут бисглицината или эквивалент, 8-12 нед, перепроверка
- **ЖДА** (Hb↓): 100-200 мг/сут разделить на 2-3 приёма, контроль через 4-6 нед
- **Беременность**: 33 мг/сут с II триместра
- ⚠️ Через день эффективнее, чем ежедневно: исследования показали, что схема «100 мг через день» абсорбируется лучше из-за феномена гепсидина

## Что улучшает / ухудшает всасывание

**Улучшают:**
- Витамин C (соотношение Fe:C = 1:2)
- Гемовое железо (мясо в том же приёме пищи)
- Кислая среда желудка (натощак или с лимоном)

**Ухудшают:**
- Танины чая, кофе (отделить от еды на 1-2 ч)
- Кальций ≥300 мг
- Фитаты (бобовые, отруби, соя)
- Полифенолы какао, вина
- Левотироксин (интервал 4 ч)
- ИПП (снижение всасывания, нужны органические хелаты)

## Лекарственные взаимодействия

| Препарат | Эффект |
|---|---|
| ИПП | ↓ всасывания — использовать бисглицинат |
| Антибиотики (фторхинолоны, тетрациклины) | хелатируют, ↓ эффект антибиотика и Fe — интервал 2-6 ч |
| Левотироксин | хелатирование, ↓ T4 — строго 4 ч |
| Аспирин, НПВП длительно | скрытые ЖКТ-кровопотери → ятрогенный дефицит |
| Кальций ≥300 мг | конкуренция за DMT-1 |
| Леводопа | хелатирование, ↓ леводопы |

Источник: `drug_interactions.md`.

## Red flags / противопоказания

- **Гемохроматоз** наследственный (HFE) — Fe категорически запрещено
- **Воспалительные процессы** (СРБ↑) — сначала разобраться с воспалением (ферритин может быть ложно нормален)
- **Повышенный ферритин с низким % насыщения** — НЕ дефицит, а АХЗ или воспаление, Fe не давать
- **Перегрузка трансфузиями** в анамнезе
- При гестационном диабете и преэклампсии — терапевтические дозы Fe без анализов противопоказаны

## Что искажает анализ ферритина

- **Любое воспаление** (СРБ↑) → ферритин ложно ↑↑↑ (острофазовый белок)
- **Поражение печени, ожирение, метаболический синдром** — ферритин ↑ без избытка Fe
- **Беременность** → гемодилюция, ложно низкие значения
- **Приём добавок Fe** — отменить за 10-14 дней перед анализом

## Алгоритм работы

1. Спросить: пол, возраст, менструации (для женщин), диета (мясо), симптомы (утомляемость, выпадение волос, ломкость ногтей, тяга к необычным продуктам)
2. Заказать: ОАК + ферритин + СРБ (обязательно вместе!)
3. Интерпретация:
   - Ферритин <30 + СРБ норма → дефицит → коррекция
   - Ферритин <100 + СРБ↑ → дефицит при воспалении → коррекция
   - Ферритин >100 без воспаления → не давать Fe
4. Коррекция: бисглицинат 25-100 мг/сут + витамин C, без чая/кофе, без кальция в один приём
5. Перепроверка через 4-6 нед (раньше — бесполезно)
6. После нормализации Hb продолжать ещё 3 мес для пополнения депо
