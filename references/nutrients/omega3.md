# Омега-3 (EPA + DHA) — cheat-sheet

## Cut-offs (Омега-3 индекс, O3i — % EPA+DHA в мембранах эритроцитов)

| O3i | Интерпретация |
|---|---|
| <4% | дефицитный, ↑ риск ССЗ |
| 4–8% | субоптимальный |
| **≥8%** | **оптимум, целевой уровень** |
| >12% | риск аддитивных кровотечений с антикоагулянтами |

O3i — золотой стандарт, но мало доступен в СНГ. Косвенно: высокая частота жирной морской рыбы в рационе.

## Lifestyle-маркеры

| Фактор | Влияние |
|---|---|
| Жирная рыба (сельдь, скумбрия, дикий лосось, сардины) <1 раз/нед | 39% имеют O3i <4% |
| Жирная рыба ≥3 раз/нед | только 20% достигают O3i ≥8% |
| Нет добавок омега-3 | риск дефицита для большинства жителей СНГ |
| Подсолнечное/кукурузное масло как основное | избыток омега-6, отношение в СНГ до 25:1 |
| Веганство/вегетарианство | конверсия ALA→EPA/DHA <5% |
| Холецистэктомия / дискинезия желчных путей | блок усвоения жиров |
| СИБР, мальабсорбция | ↓ |
| Женщины 50+ | статистически чаще дефицит |

Источник: `lifestyle_markers.md` § Омега-3.

## Формы

| Форма | Биодоступность | Когда |
|---|---|---|
| **Триглицериды (rTG, натуральная форма)** | высокая (~70% выше эфиров) | предпочтительный выбор |
| **Этиловые эфиры (EE)** | средняя | дешевле, нужно с жирной едой |
| **Фосфолипидная форма (криль)** | высокая, без рыбного послевкусия | для людей с диспепсией от рыбьего жира |
| **Растительные омега-3 (льняное, чиа, грецкий орех)** | низкая конверсия ALA→EPA/DHA <5% | НЕ замена морских |
| **Веганские из водорослей (algae oil)** | высокая | альтернатива для веганов, EPA+DHA в чистом виде |

## Дозы

- **Профилактика взрослые:** 500-1000 мг EPA+DHA/сут (не «рыбьего жира», а активных EPA+DHA!)
- **Поддержание O3i ≥8%:** 1000-2000 мг EPA+DHA/сут
- **Терапевтически** (триглицеридемия, постинфаркт): 2000-4000 мг (Ловаза, Васепа)
- **Беременность/лактация:** дополнительно 200-300 мг DHA/сут
- **Депрессия:** 1000+ мг с преобладанием EPA
- **UL** не установлен, но >3 г/сут — мониторить коагуляцию

⚠️ Соотношение EPA:DHA:
- EPA-доминирующие (для воспаления, депрессии)
- DHA-доминирующие (для мозга, беременности, развития плода)

## Сочетания и взаимодействия

- **Принимать с самой жирной едой дня** (липофильные)
- **С витамином E** для защиты от окисления
- **Хранить в холодильнике**
- **С антикоагулянтами** (варфарин, ривароксабан, дабигатран): дозы до 1 г безопасны; 3-4 г требуют контроля МНО/коагулограммы
- **Не сочетать с другими антиагрегантами** в высоких дозах

## Red flags

- **Перед операцией** — обычно отменяют за 7 дней при дозах >1 г
- **Геморрагический инсульт в анамнезе** — осторожность с высокими дозами
- **Тромбоцитопения** — высокие дозы только под контролем
- **Дефицит желчи** (холестаз) — низкая усвояемость, нужны фосфолипидные формы или мицеллированные

## Чистота и качество

- **IFOS, GOED** — стандарты чистоты, проверка на тяжёлые металлы (ртуть, кадмий, диоксины)
- **Криль** — меньший риск тяжёлых металлов (мелкая ступень пищевой цепи)
- **TOTOX** (показатель окисления) <26 — свежий продукт

## Спецгруппы

- **Беременность:** DHA 200-300 мг/сут, критично для нейрогенеза плода
- **Дети:** EPA+DHA 250-500 мг/сут для развития
- **Сердечно-сосудистые:** EPA-доминирующие (1-4 г, доказано REDUCE-IT)
- **Депрессия:** EPA-доминирующие 1000+ мг
- **Холецистэктомия:** криль или мицеллярные формы

## Алгоритм

1. Спросить: частота жирной морской рыбы, какое масло на кухне, ЖКТ-симптомы, желчный пузырь, беременность
2. Если рыба <2 раз/нед (типично для СНГ) → рекомендовать добавку
3. Заказать O3i (если доступно) или сразу профилактическая доза
4. Дотация: rTG 1000-2000 мг EPA+DHA/сут с самой жирной едой
5. Поменять масло на оливковое для коррекции соотношения омега-6/3
6. Беременным: обязательно с DHA
