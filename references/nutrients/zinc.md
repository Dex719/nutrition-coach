# Цинк (Zn) — cheat-sheet

## Cut-offs

| Маркер | Норма | Дефицит |
|---|---|---|
| Цинк сыворотки/плазмы | 10.4–16.4 мкмоль/л | <10.0 |
| Медь сыворотки | 11.0–22.0 мкмоль/л | — |
| **Соотношение Cu/Zn (CZr)** | 0.8–2.0; оптимум 0.7–1.5 | >1.5–2.0 → избыток меди / относительный дефицит Zn |
| Цинк в волосах (HTMA) | оптимум Zn:Cu = 8.2:1 – 13.2:1 | — |
| Щелочная фосфатаза (Zn-зависимый фермент) | косвенный маркер; ↓ при дефиците Zn |

⚠️ Цинк — негативный острофазовый реактант. При воспалении (СРБ↑) ложно ↓ за счёт перераспределения в печень. Медь наоборот — позитивный острофазовый, ложно ↑. Это даёт ложную картину «дефицита Zn + избытка Cu» при инфекциях, аутоиммунных обострениях.

**КОК, беременность** → повышают церулоплазмин и медь почти в 2 раза.

Источник: `lab_diagnostics.md` § 5.

## Lifestyle-маркеры

| Фактор | Влияние |
|---|---|
| Строгое веганство | дефицит у 67% (vs 11% всеядных) |
| Вегетарианство | дефицит у 51% |
| Красное мясо, субпродукты <1 раз/нед | базовый ↓ |
| Фитат/Zn >15:1 (бобовые, отруби без замачивания) | блокада абсорбции |
| Девочки-подростки 14-18 лет | пик роста + малое потребление мяса |
| Беременность без пренатальных с Zn | задержка развития |
| Хроническая диарея, лямблиоз | потери Zn |
| Регулярный алкоголь | ↑ экскреции Zn |
| Кальций + злаки/бобовые одновременно | соосаждение Zn |
| Белые пятна на ногтях, гнездная алопеция, ангулярный стоматит | ранние клинические маркеры |
| Дисгевзия (искажение вкуса), анорексия | при иАПФ → дефицит Zn |

Источник: `lifestyle_markers.md` § Цинк.

## Формы

| Форма | Биодоступность | Когда |
|---|---|---|
| **Бисглицинат** | очень высокая | универсал, мягкий для ЖКТ |
| **Пиколинат** | высокая | стандарт, особенно при кожных проблемах |
| **Цитрат** | высокая | широко доступен |
| Сульфат | средняя | дёшево, может раздражать ЖКТ |
| Глюконат | средняя | в леденцах при простуде |
| Оксид | низкая | избегать |

## Дозы

- **RDA:** мужчины 11 мг/сут, женщины 8 мг/сут
- **Профилактика:** 15-30 мг/сут элементарного Zn
- **При иАПФ, КОК, веганстве:** 15-30 мг/сут + 1-2 мг меди (соотношение 15:1)
- **Простуда** (леденцы): 75-100 мг/сут разделить, не более 5 дней
- **UL:** 40 мг/сут — длительное превышение вызывает дефицит меди и анемию

⚠️ Длительный приём Zn без меди → вторичный дефицит Cu → сидеробластная анемия + неврологические проблемы. **Соотношение Zn:Cu = 15:1** обязательно при длительном приёме.

## Тайминг и сочетания

- **Натощак или с лёгкой едой** — лучшая абсорбция
- **Утро лучше вечера** (бодрит)
- **Не сочетать в один приём с:** кальцием в высоких дозах, железом, медью (для дотации меди — отдельно)
- **Антибиотики** (фторхинолоны, тетрациклины) — интервал 2-6 ч
- **иАПФ/сартаны** — интервал 2 ч

## Лекарственные взаимодействия

| Препарат | Эффект |
|---|---|
| **иАПФ / сартаны** | хелатирование Zn в активном центре АПФ → ↑ экскреции, дисгевзия |
| **КОК** | ↑ экскреции через задержку натрия |
| **Кортикостероиды** | ↑ почечной потери |
| **Диуретики** | ↑ экскреции |
| **Антибиотики** | хелатирование |

Источник: `drug_interactions.md`.

## Red flags

- **Не принимать длительно Zn без меди** — вторичный Cu-дефицит, анемия
- **Гемохроматоз** — Zn осторожно (взаимодействие с Fe-абсорбцией)
- **Болезнь Вильсона-Коновалова** — Zn используется терапевтически (блокирует абсорбцию Cu), требует контроля
- При воспалении не интерпретировать ↓ Zn как дефицит — сначала проверить СРБ

## Алгоритм

1. Высокий риск: вегетарианство/веганство, иАПФ/сартаны, КОК, девочки-подростки, дисгевзия, белые пятна на ногтях
2. Заказать: Zn + Cu сыворотки + СРБ (исключить ложь от воспаления)
3. Если CZr >1.5 + клиника — дотация Zn
4. Дотация: пиколинат или бисглицинат 15-30 мг/сут + 1-2 мг меди
5. Не больше 3 месяцев непрерывно без контроля Cu
6. Перепроверка через 2-3 месяца
