# Протоколы по группам пациентов

> Быстрые сводки по основным нутритивным потребностям для разных групп. Детали — в `special_populations.md` (полный отчёт по группам) и в `references/nutrients/*` (по каждому нутриенту).

---

## Беременность (по триместрам)

### Прегравидарная подготовка (за 3 месяца до зачатия)

| Нутриент | Доза | Цель |
|---|---|---|
| L-метилфолат | 400-800 мкг/сут | профилактика ДНТ, сниж. риска на 70% |
| Йод (KI) | 150-250 мкг/сут | накопление в ЩЖ |
| D3 | 1000-2000 МЕ/сут | депо |
| Железо (бисглицинат) | 25 мг/сут при ферритине <50 | пополнение депо до беременности |
| Омега-3 (DHA) | 200-300 мг/сут | подготовка к нейрогенезу плода |
| B12 (метилкобаламин) | 250-500 мкг/сут | особенно если веганка |

### I триместр

- Метилфолат **600-800 мкг/сут** (РФ 600, нормативы КЗ 800 для подростков)
- KI 220-250 мкг/сут
- D3 1000-2000 МЕ/сут
- Холин 450 мг/сут (часто упускается!)
- Магний 350-400 мг/сут при крампи/тонусе

### II–III триместр (добавить)

- Железо **33 мг/сут** (или больше при ферритине <30) — со II триместра
- DHA 200-300 мг/сут
- Кальций 1000-1300 мг/сут

### Лактация

- KI **290 мкг/сут**
- D3 1500-2000 МЕ/сут (или для младенца отдельно)
- Холин 550 мг/сут
- DHA 200-300 мг/сут
- B12 500 мкг/сут особенно у веганок

### Red flags при беременности

- **Витамин A** в форме ретинола >3000 МЕ — тератоген. Только бета-каротин!
- Высокие дозы йода (>200 мкг) при некомпенсированном узловом зобе
- Терапевтические дозы Fe без ферритина + СРБ (риск гестац. диабета, преэклампсии)
- Бесконтрольные мегадозы любых нутриентов

Подробности: `special_populations.md` § 1.

## Дети (0–18 лет)

### 0–1 год (грудные)

| Нутриент | Доза | Источник |
|---|---|---|
| D3 | **1000 МЕ/сут** с 4-й недели | капли, обязательно |
| Железо | прикорм с 4-6 мес мясные пюре | если ГВ дольше — мониторинг ферритина |
| DHA | 100 мг/сут | из ГВ или смеси с DHA |
| K1 | при рождении (роддом) | профилактика геморр. болезни новорождённых |

### 1–3 года

| Нутриент | Особенность |
|---|---|
| Железо 10 мг/сут | критично; ограничивать коровье молоко (<500 мл/сут) |
| D3 1000 МЕ/сут | продолжать |
| Йод 90 мкг/сут | через йодиров. соль или капли |

### 3–7 лет

- Кальций 900-1000 мг/сут (молочные)
- Йод 120 мкг/сут
- D3 1000 МЕ/сут

### 7–12 лет

- Йод 120-150 мкг/сут (когнитивное развитие!)
- Цинк (особенно девочки)
- Омега-3 250-500 мг EPA+DHA

### Подростки

- Фолат **800 мкг/сут** (по нормативам РК) — пубертатный скачок роста
- D3 1000-1500 МЕ/сут (распространённость дефицита 71%)
- Железо (особенно девочки с менархе): 18 мг/сут
- Кальций 1300 мг/сут (пиковая костная масса)

### Red flags у детей

- D3 >2000 МЕ/сут без анализа 25(OH)D + кальциурии (нефрокальциноз)
- Железо без верифицированного дефицита (подавляет цинк, питает патогенов)
- Витамин A в высоких дозах

Подробности: `special_populations.md` § 2.

## Пожилые (60+)

Главные проблемы:
1. Атрофический гастрит → B12 (даже без метформина/ИПП)
2. Снижение синтеза D в коже в несколько раз
3. Полипрагмазия → лекарственно-индуцированные дефициты
4. Саркопения → белок + витамин D + омега-3

### Базовый протокол

| Нутриент | Доза | Обоснование |
|---|---|---|
| D3 | 1000-2000 МЕ/сут (800 МЕ по РФ норме) | синтез в коже ↓↓↓ |
| B12 (метил) | 500-1000 мкг/сут | гипоацидность |
| Магний (бисглицинат) | 300-400 мг/сут | часто ИПП, диуретики |
| Омега-3 | 1000 мг EPA+DHA/сут | саркопения, когнитивная защита |
| Белок | 1.2-1.6 г/кг/сут | профилактика саркопении |

### Если на полипрагмазии — проверять:

- ИПП → B12, Mg, Ca, Fe (см. `drug_interactions.md` § 1)
- Метформин → B12, фолат (3-5 лет терапии = дефицит)
- Статины → CoQ10, D, K2 (см. § 3)
- Диуретики → K, Mg, Zn, B1 (см. § 6)

### Red flags

- D >2000 МЕ без 25(OH)D
- Болюсы D любые (запрещены после 50!)
- Высокие дозы Ca без D и K2 (риск кальцификации сосудов)
- Железо без ферритин+СРБ (часто АХЗ маскируется под ЖДА)

Подробности: `special_populations.md` § 3.

## Веганы / строгие вегетарианцы

### Обязательные добавки

| Нутриент | Доза | Почему |
|---|---|---|
| **B12 (метилкобаламин)** | 1000 мкг 1-2 раза/нед или 250 мкг/день | в растениях НЕТ |
| **D3 (веганский, из лишайника)** | 1000-2000 МЕ/сут | мало источников |
| **DHA из водорослей (algae)** | 200-300 мг/сут | конверсия из ALA <5% |
| Йод (KI) | 150 мкг/сут | если нет морских водорослей |
| Железо (бисглицинат) | 25 мг/сут женщины + контроль | негемовое усваивается 2-8% |
| Цинк (пиколинат) | 15-30 мг + 1-2 мг Cu | фитаты блокируют |
| Кальций | 1000 мг через обогащ. растит. молоко или добавки | если нет молочных |

### Лабораторный мониторинг для веганов раз в год

- Ферритин + СРБ
- B12 общий + гомоцистеин + по возможности HoloTC
- 25(OH)D
- Цинк + медь
- Йод (или ТТГ + Т4св)

Подробности: `special_populations.md` § 4.

## Спортсмены (выносливость / силовые)

### Выносливость (бег, велоспорт, плавание)

| Нутриент | Особенность |
|---|---|
| Железо | целевой ферритин **>50** (не >30 как у обычных); риск спорт-анемии |
| Натрий, калий, магний | электролиты при тренировках >1 ч (Mg цитрат) |
| D3 | 2000-4000 МЕ/сут (профилактика травм, иммунитет) |
| Омега-3 | 2-3 г EPA+DHA (восстановление) |
| Антиоксиданты (C, E) | умеренно, избыток снижает адаптацию к тренировкам |

### Силовые

| Нутриент | Особенность |
|---|---|
| Белок | 1.6-2.2 г/кг/сут |
| Креатин моногидрат | 3-5 г/сут (доказательная база огромная) |
| D3 | 2000-4000 МЕ |
| Магний (треонат для нервной системы или цитрат) | 400-600 мг |
| Цинк | 15-30 мг (синтез тестостерона) |

Подробности: `special_populations.md` § 5.

## Менопауза / Перименопауза

### Главное

| Нутриент | Доза |
|---|---|
| **Кальций** | 1200-1500 мг/сут (цитрат предпочтительно) |
| **D3** | 1000-2000 МЕ/сут |
| **K2 (МК-7)** | 90-180 мкг/сут |
| Магний | 300-400 мг/сут |
| Омега-3 | 1000-2000 мг EPA+DHA |
| Фитоэстрогены (соя, льняное) | пищевые источники по толерантности |

### Не забыть

- Денситометрия костей при отягощённом анамнезе
- Контроль 25(OH)D, ферритина, B12
- Если статины — CoQ10
- Если бисфосфонаты — Ca + D обязательно

Подробности: `special_populations.md` § 7.

| Нутриент | Доза |
|---|---|

ОАК, ферритин+СРБ, B12+HoloTC+MMA, 25(OH)D, кальций+ПТГ, цинк+медь, фолат

## Гипотиреоз / Аутоиммунный тиреоидит (Хашимото)

### Базовый протокол

| Нутриент | Доза | Цель |
|---|---|---|
| **Селенометионин** | **200 мкг/сут** 3-6 мес | снижение АТ-ТПО |
| Йод (KI) | ОСТОРОЖНО, только при подтверждённом дефиците + ВСЕГДА с Se | риск обострения при изолированной дотации! |
| D3 | 2000-4000 МЕ + контроль | целевой уровень 50-60 |
| Железо | целевой ферритин >70-100 (при тиреоидной дисфункции выше норма) |
| Цинк | 15-30 мг |
| Магний | 400 мг |

### Лекарственные нюансы

- **Левотироксин строго натощак** за 30-60 мин до еды, запивать водой
- **Кальций, Fe, Mg** — интервал 4 ч от левотироксина
- ИПП (омепразол) снижают абсорбцию левотироксина (нужны более высокие дозы)
- Соевые продукты — снижают абсорбцию

Подробности: `special_populations.md` § 10, `iodine.md` (раздел Йод+Se синергия).

## Диабет 2 типа / Инсулинорезистентность

### При метформине обязательно:

- B12 метилкобаламин **1000 мкг/сут** перорально (после 2-3 лет терапии — обязательно, у всех)
- Метилфолат 400-800 мкг/сут (вторичный дефицит через B12)
- **Магний** (цитрат или бисглицинат) **400 мг/сут** — OR дефицита при СД2 = 4.62
- D3 2000-4000 МЕ
- Омега-3 1-2 г EPA+DHA
- Альфа-липоевая кислота 600 мг/сут (при нейропатии)
- Хром 200-400 мкг

Подробности: `special_populations.md` § 12, `drug_interactions.md` § 2 (метформин).
